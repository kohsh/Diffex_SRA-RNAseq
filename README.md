# ✒️ SRA-Diffex-analysis

Our Guthub repo include pipeline for running Diferential gene expression analysis on [SRA](https://www.ncbi.nlm.nih.gov/sra) fastq files using [Boruta](https://www.jstatsoft.org/article/view/v036i11) tool. The pipeline is written in an [R](https://github.com/rstudio/rstudio) programming language.
